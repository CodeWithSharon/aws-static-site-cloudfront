# AWS Static Site — S3 + CloudFront

Personal portfolio hosted on AWS S3 with CloudFront CDN.

## Live Site
https://dd2etqvmxlvc1.cloudfront.net

## Architecture
Browser → CloudFront (HTTPS) → S3 (Static Hosting)

## Services Used
- Amazon S3 (static website hosting)
- Amazon CloudFront (CDN + HTTPS)
- IAM bucket policies
